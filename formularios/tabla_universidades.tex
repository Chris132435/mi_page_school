\vspace{0.5cm}

\section*{Diferencias Clave}

\begin{itemize}
\item \textbf{UNAM:} Examen teórico con alta competencia; requiere alto puntaje para ingeniería.
\item \textbf{IPN:} Mayor dificultad en áreas científicas; enfoque técnico.
\item \textbf{UAM:} Evaluación equilibrada entre conocimientos y razonamiento; influye el promedio.
\item \textbf{TEC de Monterrey:} Proceso integral que considera aspectos académicos y personales.
\end{itemize}

\vspace{0.5cm}

\section*{Conclusión}

\begin{itemize}
\item Más exigente en ciencias: IPN
\item Mayor competencia: UNAM
\item Evaluación equilibrada: UAM
\item Enfoque integral: TEC de Monterrey
\end{itemize}
